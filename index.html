📥 Guide d'importation

Format JSON (recommandé)

Exportez d'abord vos données avec le bouton "📤 Exporter" pour voir le format exact.

[
  {
    "formateur": "Marie Dupont",
    "formations": ["Excel", "Word"],
    "type": "bureautique",
    "tarif": "450€/jour",
    "dossiers": 25,
    "note": 4.8,
    "contrat": "cdi",
    "meet": "https://meet.google.com/...",
    "linkedin": "https://linkedin.com/in/...",
    "observations": "Excellente formatrice"
  }
]

Format CSV

Utilisez un fichier CSV avec ces colonnes (séparateur: virgule, point-virgule ou tabulation) :

formateur,formations,type,tarif,dossiers,note,contrat,meet,linkedin,observations
Marie Dupont,"Excel,Word",bureautique,450€/jour,25,4.8,cdi,https://meet.google.com/...,https://linkedin.com/in/...,Excellente formatrice
Jean Martin,"Anglais,Espagnol",langues,60€/h,18,4.5,sous-traitant,,,Spécialisé anglais pro

💡 Astuce : Les colonnes peuvent avoir des noms légèrement différents (nom, formateur, prix, tarif, etc.)
